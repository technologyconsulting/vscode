    ```sh

    ```
  - [x] Git, Initialise local repo:
    ```sh
    git init
    ```
  - [x] Git, Rename local repo branch from master to main:
    ```sh
    git branch -m main
    ```
  - [x] Git, Stage Changes:
    ```sh
    git add .
    ```
  - [x] Git, Commit Changes:
    ```sh
    git commit -m "chore: initial upload"
    ```
  - [x] GitHub, Create new repo on GitHub and push local repo to GitHub:
    ```sh
    gh repo create
    ```
